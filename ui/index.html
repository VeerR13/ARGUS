System Capabilities

Three Layers.
One Pipeline.

SYS-01
Detection
RF-DETR (Roboflow Detection Transformer) runs frame-by-frame inference, localizing and classifying vehicles with confidence thresholds tuned for urban traffic density.
RF-DETR · COCO-80
SYS-02
Tracking
ByteTrack assigns persistent IDs across frames using a two-stage matching strategy — high-confidence detections first, then low-confidence re-association to minimise ID switches in occlusion.
ByteTrack · IOU + ReID
SYS-03
Analytics
Aggregated track data flows into a structured event log — vehicle counts, class distribution, speed bands, and anomaly flags — exposed via REST API for downstream dashboards.
REST API · JSON Events
